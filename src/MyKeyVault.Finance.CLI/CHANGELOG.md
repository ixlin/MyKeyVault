# 更新日志 v1.0.1

## 🔧 修复内容

### 1. 修复重复记录问题
**问题描述**：
- 导出年报时，现金流量表出现两列相同的数据（记录1和记录2）
- API 返回的数据可能包含重复记录

**解决方案**：
- 添加自动去重逻辑，基于完整的 JSON 内容进行去重
- 确保每个报表只显示唯一的数据记录

**技术实现**：
```csharp
var uniqueDataArray = dataArray
    .GroupBy(item => item.GetRawText())
    .Select(group => group.First())
    .ToList();
```

### 2. 优化列标题显示
**修改前**：
- A 列：`字段`
- B 列：`记录1`
- C 列：`记录2`（如果有多条记录）

**修改后**：
- A 列：`报表项目`
- B 列：`项目值`（单条记录时）或 `项目值1`（多条记录时）
- C 列：`项目值2`（如有）

**智能适配**：
- 当只有一条记录时，列标题显示为"项目值"（更简洁）
- 当有多条记录时，列标题显示为"项目值1"、"项目值2"...（便于区分）
